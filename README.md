# Sistema de Gestión de Admisiones - Liceo PJMA

Este proyecto es una solución automatizada desarrollada por **Edison Clase** para optimizar el proceso de admisión escolar. El sistema integra tecnologías de nube para asegurar una comunicación rápida, segura y profesional con los padres y tutores.

## Funcionalidades Principales
- **Automatización Completa:** Envío masivo de correos electrónicos con requisitos de inscripción.
- **Seguridad de Datos:** Cumplimiento con la **Ley No. 172-13** sobre protección de datos personales en la República Dominicana.
- **Integración con Microsoft 365:** Lectura en tiempo real de formularios alojados en SharePoint/Excel.
- **Notificación Multi-canal:** Incluye enlaces directos a grupos de seguimiento en WhatsApp.

## 🛠️ Especificaciones Técnicas
- **Lenguaje:** Python 3.14.3.
- **Infraestructura:** Ejecución en la nube mediante **GitHub Actions**.
- **Seguridad:** Uso de variables de entorno cifradas (GitHub Secrets) para proteger credenciales institucionales.
- **Logs de Auditoría:** Registro histórico de todos los correos enviados para fines de control interno.
